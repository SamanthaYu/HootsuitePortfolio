Paint Program: Java

I created this Paint Program in Java for my AP Computer Science 12 class and is built around the concept of double buffering. When I implemented the layers, I had to change how I was approaching the double buffering concept. My Paint Program went beyond the minimum requirements by using classes and concepts that were not taught in the course.

Blackjack Game: JavaScript

I created a Blackjack game in JavaScript for my ICT: Modular Survey 11 class. In my version of Blackjack, the player can select their bet from pre-determined values, going all in, or entering in their own amount. I considered several possible outcomes, such as the user entering invalid amounts, pressing "cancel", having less money than the standard bets, or when the player goes bankrupt.
