Startup

Since PaintApp.java has a Frame and is a WindowListener, MouseListener, and MouseMotionListener, the program must set the size of the applet, add Event listeners, and make the applet visible on startup. When createApp() is called, the "click-able" components like the menu, sidebar and canvas are created. These "click-able" components cannot be viewed yet until paint() is called.
